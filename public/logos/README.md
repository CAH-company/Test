# Folder na loga klientów

Wrzuć tutaj pliki z logotypami klientów używając poniższych nazw plików.
Obsługiwane formaty: `.svg`, `.png`, `.webp`, `.jpg`

## Oczekiwane nazwy plików

| Klient          | Nazwa pliku                    |
|-----------------|-------------------------------|
| Benefit Systems | `benefit-systems.svg` (lub `.png`) |
| NEUCA           | `neuca.svg`                   |
| OnlyBio         | `onlybio.svg`                 |
| E. Wedel        | `e-wedel.svg`                 |
| Cordia          | `cordia.svg`                  |
| Marsh           | `marsh.svg`                   |
| Stonex          | `stonex.svg`                  |
| Invisibobble    | `invisibobble.svg`            |

## Wskazówki

- Loga będą automatycznie wyśrodkowane i przeskalowane do maksymalnej wysokości **48 px** przy zachowaniu proporcji.
- Zalecany format: **SVG** (wektorowy, skaluje się bez utraty jakości).
- Jeśli używasz PNG/WebP, zalecana szerokość: min. **200 px** na przezroczystym tle.
- Pliki placeholder (`.placeholder.svg`) można usunąć po dodaniu prawdziwych logotypów.
